## 📖 What is bugSkills?

bugSkills helps you convert HackerOne vulnerability reports into reusable AI skills. This makes it easier to analyze, organize, and apply the information from your reports in an automated or AI-driven setting. You don’t need programming knowledge to use this software. It comes with a user-friendly interface that guides you through each step.

## 🛠️ Troubleshooting and Support

If you encounter problems:

- **Installer Won’t Open:** Check if Windows blocked the file. Right-click the installer > Properties > Unblock if needed.
- **App Crashes:** Restart your computer and try again.
- **Reports Not Loading:** Confirm the file format is supported.
- **Conversion Errors:** Make sure the report contains clear text. Scanned images may not work.
